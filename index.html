IoT-Based Desk Occupancy Detector

Smart Workspace Management System

Abstract Introduction Problem Methodology Architecture Working Results Future Scope Team

Abstract

This project presents an IoT-based desk occupancy detection system for real-time workspace monitoring. Using an ultrasonic sensor and NodeMCU, the system detects human presence and displays occupancy status remotely while maintaining privacy.

Introduction

The Internet of Things (IoT) enables devices to communicate and exchange data seamlessly. Monitoring desk usage is essential for optimizing space and energy usage in smart campuses and offices.

Problem Statement & Motive

Existing desk monitoring systems are manual, expensive, or invasive. This project aims to build a low-cost, privacy-safe and automated solution using ultrasonic sensing.

Methodology

  • Ultrasonic sensor measures distance
  • NodeMCU processes sensor data
  • Status sent via Wi-Fi
  • Occupied / Unoccupied displayed

System Architecture

Ultrasonic Sensor → NodeMCU → Wi-Fi → Cloud → Web Interface

Working Principle

If the measured distance is less than 60 cm, the desk is considered occupied. If the distance exceeds 75 cm, the desk is marked unoccupied. LED indicators provide local feedback.

Results

  • Accurate occupancy detection
  • Low power consumption
  • Real-time monitoring

Future Scope

  • Multi-desk monitoring
  • Smart lighting & HVAC control
  • AI-based usage analysis
  • Smart campus deployment
