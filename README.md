# IoT-Based Desk Occupancy Detector

This project is a smart IoT-based system to detect desk occupancy using
ultrasonic sensors and NodeMCU ESP8266.

## Features
- Privacy-safe occupancy detection
- Real-time monitoring
- Low-cost IoT solution
- Smart campus ready

## Technologies Used
- Arduino (C)
- HTML, CSS, JavaScript
- IoT Sensors
